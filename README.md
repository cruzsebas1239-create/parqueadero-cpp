# Sistema de Parqueadero

Nombre: Diego Sebastián Cruz Suárez
        Horus Reyes

## Descripción
Sistema desarrollado en C++ que simula la gestión de un parqueadero utilizando estructuras dinámicas.

## Funcionalidades
- Registro de entrada y salida de vehículos
- Cálculo de tarifa por tiempo
- Visualización del mapa
- Control de disponibilidad

## Innovaciones implementadas

### Gestión dinámica de tarifas
Cálculo del valor basado en el tiempo de permanencia del vehículo.

### Validación de rutas
El sistema garantiza que los espacios sean accesibles mediante vías dentro del mapa.

### Tipos de vehículos diferenciados
El diseño permite escalar el sistema para manejar distintos tipos de vehículos.

### Arquitectura modular
Uso de funciones para organizar el código y facilitar mantenimiento.

## Tecnologías usadas
- C++
- Punteros
- Matrices
- Funciones
- Ciclos for
